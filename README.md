## Data Science Using Python
## NSIC Internship Project (Academic Year 2022–23)
## Internship Details

*Organization*: National Small Industries Corporation Ltd. (NSIC)

*Project Title*: Data Science Using Python

*Report Type*: Internship

*Duration*: 17 July 2023 – 31 July 2023

*Academic Year*: 2022–23

*Department*: Computer Science & Engineering (AI & ML)

## Project Overview

This repository documents the work carried out during the Data Science Using Python internship at NSIC.
The internship focused on building a strong foundation in data science concepts, Python programming, and machine learning techniques, with an emphasis on hands-on implementation and real-world data analysis.

## Technologies & Tools Used

*Programming Language*: Python

*Libraries*: NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn

*Concepts*: Data Analysis, Data Visualization, Machine Learning, Clustering

*Tools*: Jupyter Notebook / Python IDE

## Work Performed

- Learned and implemented Python fundamentals including data types, control structures, and functions.

- Performed data cleaning, preprocessing, and transformation on real-world datasets.

- Built Python-based automation pipelines to streamline large-scale data analysis.

- Conducted Exploratory Data Analysis (EDA) to uncover patterns, trends, and insights.

- Visualized data using Matplotlib and Seaborn for effective insight communication.

- Developed and refined clustering models to extract actionable insights.

- Applied machine learning techniques for data-driven decision-making.

- Worked on real-life data science use cases, strengthening analytical and problem-solving skills.

## Key Learning Outcomes

- Strong understanding of data science workflows using Python

- Practical experience with data preprocessing and visualization

- Hands-on exposure to machine learning and clustering algorithms

- Improved analytical thinking and problem-solving abilities

- Experience in translating data insights into actionable decisions

##  Conclusion

This internship at National Small Industries Corporation Ltd. (NSIC) served as a valuable learning experience by bridging theoretical knowledge with practical implementation.
It played a significant role in strengthening my foundation in Data Science and Machine Learning using Python during the 2022–23 academic year.
